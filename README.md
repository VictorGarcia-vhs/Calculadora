# Calculadora
Implementação de uma calculadora em aplicação movel 

Levantamento de requisitos base do sistema Calculadora

- O Sistema deve ter operações base obrigatorias (*,/,-,+)
- O Sistema deve ter operação Raiz Quadrada obrigatoria (√)
- O Sistema deve ter uma tonalidade escura e gradiente
